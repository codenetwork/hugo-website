# macOS setup
1. Install [Homebrew](https://brew.sh)

2. Install Git, Hugo, and uv from Homebrew
```zsh
brew install git hugo uv
```

3. Clone and enter the project
```zsh
git clone https://github.com/codenetwork/hugo-website.git
cd hugo-website
```

4. Start the Hugo web server and view the site at the URL displayed in your terminal.
```zsh
hugo server
```
